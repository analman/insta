# Instagram scraper
![Go](https://github.com/lanzay/insta/workflows/Go/badge.svg)

Scrape photos from Instagram without authorisation

Scrap by: 
* Users names
* List of users names
* Tag

```
insta -u nasa,kyliejenner -f list.txt -t topmodel foto
```
